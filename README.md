# OneHealth DataSpace

Landing page for the promotion and communication of the OneHealth DataSpace project.

## Description

OneHealth DataSpace is a platform that enables secure, ethical, governed, and interoperable sharing and access to health sector data, while respecting data sovereignty. It is focused on the One Health domain, meaning it integrates human, animal, and environmental health data within a single space, with the aim of providing a comprehensive and unified view. This facilitates joint data analysis, improving decision-making, research, and responses to complex health challenges.

Through this landing page, the platform is presented and promoted, while also serving as a gateway to its social media channels and as a communication channel for organizations interested in joining or receiving training.

## Features

- Overview of the project and its key related concepts  
- Project information, including funding and participating entities (displayed in the footer)  
- Modern, responsive interface aligned with the project's brand identity  
- Access point to social media  
- Access to training videos  
- News and events section related to the project  
- Frequently Asked Questions (FAQ) section covering the most relevant topics  
- Forms for entity onboarding and training registration  
- Information on how to apply for the Data Spaces Kit funding program  
- Section dedicated to CESGA and the project team  
- Multilingual platform (Galician, Spanish, and English)  
- Use cases and data catalog designed to facilitate the onboarding of new entities
